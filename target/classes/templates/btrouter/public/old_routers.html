The + sign here will allow the user to add a new option to the checkboxes. This will add it as an option for all existing routers, though will be unchecked by default.

The + sign here represents "Add", so as to add a new Router Type. When clicked, the dropdown box below will become a text field to allow text input. This is the only fully interactive/working + sign in the prototype, so use this to see how it'll work.

This page should represent all the routers that BT uses and can install. The full configuration options for the router should be what is used here. Specific router limitations or options will be introduced on a per-customer basis.


So for example, you should list the maximum number of ethernet ports that the router supports, and then if you want, say, a water company to only ever be able to select 1 as the option for the ethernet port (even if the possible maximum is 99 or whatever), that can be configured in the customers page.

All of this form is interactive except the bottom 2 + signs, the profile icon, and the save and delete buttons.


For an example of how it works, select a Virtual Access router. Notice how Serial is never an option for these. Now select the Merlin router, and notice how Serial becomes an option.

+ signs will be used to collapse the categories instead. Dedicated "add x" buttons will be added instead.
